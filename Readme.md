Simple Chat App

A lightweight real-time chat application built with modern web technologies. This app allows users to send and receive messages instantly in a clean and responsive interface.

🚀 Features
🔐 User authentication (login/signup)
💬 Real-time messaging
📩 Message status (Sent, Delivered, Seen)
🧑‍🤝‍🧑 One-to-one conversations
📱 Responsive UI (mobile + desktop)
⚡ Fast and minimal design
🛠️ Tech Stack

Frontend

Next.js (React + TypeScript)
 CSS

Backend

Node.js  (or your backend)
WebSockets / Socket.IO (for real-time communication)
